# Reserve VA Offset Calculator (Starter)

Mobile-friendly (PWA-style) web calculator to estimate VA offset/recoupment vs Reserve drill/AT pay.

## MVP.
Inputs:
- VA monthly ($)
- Military monthly base pay ($)
- UTAs (drill periods)
- Active duty days (AT/ADOS/etc.)

Outputs:
- VA daily, Military daily
- Waiver days (UTAs + AD days)
- Estimated VA recoupment
- Estimated military gross (base pay)
- Net advantage and recommendation

## Deploy (no installs)
Import this repo into Vercel and set Root Directory to `apps/web-pwa`.
